This project was my second to last project for Year 1 and was title "Mini Project" due to the usage of a micro-controller. The idea of this project was for a user to be able to press a button in front of a carwash, which then prompts a nearby camera to scan the license plate of the car from where it checks if the said user is subscribe to the carwash services and whether if he is or not it will either allow him to enter or to register. For this project I used a micro-controller titled ESP8266 and the linux system Raspberry Pi 3 and a MQTT brocker. The first thing I did for this project was set up the micro-controller so that it can be subscribe to a specific MQTT topic and also publish in the same topic. This way it can activate the whole system that is managed by the Raspberry Pi 3 and do a specific task when a specific command is published. Here is the code for the ESP8266 I wrote: Code.

Afterwards I wrote the code for the Raspberry Pi 3 that enables the linux system to be able to be subscribe to the same MQTT brocker as the ESP8266 and to publish there too. That allows it to give specific commands to the micro-controller and also know when to start the checking proccess. The proccess itself consits of taking a picture of the license plate and then sending it to the Google Vision API, which gives back the letters and numbers of the plate. From there the Raspberry Pi 3 checks to see if the data is in the IoT Database (Google Database) and then depending on whether the users in the database or not the linux system sends a specifc command to the micro-controller.
