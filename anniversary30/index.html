This project was a more simple one than the other two due to its simplistic idea of representing my former school using HTML , CSS, PHP and some Javascript. The site itself was made in a way that it can be dynamically updated without messing up the whole style of website. This is done by seperating the site into a few php pieces where each corresponds to a specific part of the website. Furthermore, the project contains information about the school and a functioning javascript gallery where a user can easily navigate and see the different pictures connected to the project and the school itself. While the website in itself doesnt have a grand idea behind it still reached the National Olympiad of 2016.
